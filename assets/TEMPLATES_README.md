# Gsharp.Templates

`dotnet new` project templates for the [GSharp programming language](https://github.com/DavidObando/gsharp).

## Quick Start

```bash
# Install the templates
dotnet new install Gsharp.Templates

# Create a new GSharp console app
dotnet new gsharp-console -n MyApp
cd MyApp
dotnet run
```

## Included Templates

| Template            | Short Name       | Description                     |
|---------------------|------------------|---------------------------------|
| GSharp Console App  | `gsharp-console` | A console application in GSharp |

## Requirements

- .NET 10.0 SDK or later.
- [Gsharp.NET.Sdk](https://www.nuget.org/packages/Gsharp.NET.Sdk) (automatically referenced by the generated project).
