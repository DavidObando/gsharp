# A Tour of G#

Welcome to a quick tour of G#. Each chapter introduces one small part of the language with short examples drawn from the repository's checked-in samples.

G# is a modern .NET language with Go-, Kotlin-, and Swift-style ergonomics. You will see packages, imports, `func`, top-level statements, structs, slices, maps, arrow lambdas, `if let`/`guard let`, structured concurrency with `scope`, and direct calls into CLR libraries.

## Chapters

1. [Basics](/docs/tour/basics) — variables, constants, functions, printing, and primitive types.
2. [Types and values](/docs/tour/types) — structs, data structs, data classes, classes, arrays, slices, maps, and zero values.
3. [Control flow](/docs/tour/control-flow) — `if`, `switch`, `for`, `while`, `if let`/`guard let`, and pattern matching.
4. [Concurrency](/docs/tour/concurrency) — `scope`, `async`/`await`, and `async sequence[T]`.
5. [.NET interop](/docs/tour/dotnet-interop) — imports, CLR types, extension functions, LINQ, and events.

If you have not installed the tools yet, start with [Install G#](/docs/getting-started/install). If you want the shortest possible first program, use [Quickstart: Hello, G#](/docs/getting-started/quickstart).
