# GSharp library with xUnit test project

Generated by `dotnet new gsharp-xunit`.

## Layout

```
GsharpLibrary.sln
GsharpLibrary/
  GsharpLibrary.gsproj        GSharp class library (Gsharp.NET.Sdk)
  Greeter.gs                  public Greeter class
GsharpLibrary.Tests/
  GsharpLibrary.Tests.gsproj  GSharp xUnit project, ProjectReference -> GsharpLibrary
  GreeterTests.gs             xUnit tests written in GSharp
```

Both projects are written in GSharp. The test project is an ordinary GSharp library (`Gsharp.NET.Sdk`) flagged with `<IsTestProject>true</IsTestProject>` and referencing `Microsoft.NET.Test.Sdk`, `xunit`, and `xunit.runner.visualstudio`. The tests apply xUnit attributes with GSharp's `@`-attribute syntax (`@Fact`, `@Theory`, `@InlineData`); gsc emits them as the corresponding xUnit attributes, so the standard VSTest runner discovers and runs them.

## Build & test

```sh
dotnet build
dotnet test
```
